# 📚 Book Tracker

O **Book Tracker** é uma aplicação web interativa desenvolvida com Streamlit para organizar, explorar e analisar dados de leitura de forma intuitiva e visual.

A aplicação permite registrar livros, filtrar informações e gerar análises sobre hábitos de leitura, transformando dados pessoais em insights claros e úteis.

---

## 🎯 Objetivo

Aplicar conceitos de análise de dados em um contexto real, permitindo:

- Organização de uma biblioteca pessoal  
- Exploração interativa dos dados  
- Identificação de padrões de leitura  

---

## ⚙️ Funcionalidades

### 📖 Biblioteca
- Visualização completa dos livros cadastrados  
- Filtro por gênero  
- Filtro por nota mínima  
- Ordenação por avaliação  

### 📊 Estatísticas
Visualização com indicadores de:

- Total de livros lidos  
- Média geral das avaliações  
- Gênero mais lido  
- Autor(a) mais frequente  

Gráficos:
- Distribuição de livros por gênero  
- Média de notas por gênero  
- Quantidade de livros por ano  
- Modo de leitura (físico, Kindle, audiobook)  

### 🔮 Recomendador
Sistema simples de recomendação baseado em:

- Gênero selecionado  
- Ranking por maior nota  

---

## 🧠 Diferencial

O projeto vai além de um simples registro de livros, trazendo uma camada analítica que permite:

- Identificar padrões de comportamento  
- Avaliar preferências literárias  
- Explorar dados de forma interativa  

---

## 🛠️ Tecnologias Utilizadas

- Python  
- Streamlit  
- Pandas  
- Matplotlib  

---

## 🗂️ Estrutura dos Dados

Os dados são organizados em formato tabular com as seguintes informações:

- Título  
- Autor  
- Gênero  
- Nota  
- Ano e mês de leitura  
- Modo de consumo  
- Tempo de leitura
